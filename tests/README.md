# Tests

This directory contains smoke tests plus a small number of fixed fixtures.

## Generated Artifacts

Most `tests/tmp_vcs_*` directories are generated by smoke tests and should not be committed. They can be removed with:

```powershell
npm run clean:test-artifacts
```

Preview the cleanup without deleting anything:

```powershell
npm run clean:test-artifacts -- --dry-run
```

The cleanup script preserves `tests/tmp_vcs_p3_render_smoke` because it contains tracked fixture images from older tests.

Generated cache/build outputs such as `.cache_video_create_v5`, `.video_create_project`, `chunks`, `build_report.json`, and `concat_list.txt` are ignored.
